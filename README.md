# Chatleaf

Ask question to your documents locally using open-source LLMs. 

## Getting Started

### Install dependencies

```bash
poetry shell
poetry install
```

### Run the app

```bash
chainlit run chatleaf/app.py
```

## Tech Stack

- [Ollama](ollama.ai)
- [Chainlit](chainlit.io)

## Supported

- [x] Plain text
- [x] Markdown
- [ ] PDF
- [ ] Docx
- [ ] CSV
- [ ] URLs
